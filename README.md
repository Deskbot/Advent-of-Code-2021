# Advent of Code 2021

## Days: 1-3

Language: Zig

## Run

```
zig build && ./zig-out/bin/2021
```

## Test

```
zig test src/main.zig --main-pkg-path src
```

## Days: 4

Language: V

## Run

```
./v/v src/main.v -o v-out/main && ./v-out/main
```

## Days: 4-

Language: Nim

## Run

```
nim compile -o:nim-out/main --run src/main.nim
```
